# 🌿 NatureLens – AI-Powered Plant Care Mobile App

NatureLens is an AI-powered mobile application designed to help home gardeners identify plants, detect diseases, and improve plant care. The app integrates artificial intelligence with a simple and user-friendly interface to make gardening easier and more accessible.

---

## 🚀 Features

- 📸 AI-Based Plant Identification  
- 🦠 Plant Disease Detection  
- 🌦 Weather-Based Alerts  
- 💧 Watering Reminders  
- 📔 Personal Plant Growth Journal  
- 📚 Digital Herb Library  
- 🌍 Multilingual Support (English & Urdu)  
- 📱 User-Friendly Mobile Interface  
- 🗄 Backend Integration with PostgreSQL  

---

## 🛠 Tech Stack

Frontend:
- React Native
- Expo

Backend:
- Django
- PostgreSQL Database
- REST APIs

# ⚙️ HOW TO RUN THIS PROJECT

## 1️⃣ Install Node.js
Download and install from:
https://nodejs.org

---

## 2️⃣ Clone the Repository

git clone https://github.com/zohrakhanG/NatureLens.git

cd NatureLens

---

## 3️⃣ Install Dependencies

npm install

---

## 4️⃣ Start the Expo Development Server


npx expo start

---

# 📱 RUN ON PHONE

### Step 1:
Download **Expo Go** app 

### Step 2:
